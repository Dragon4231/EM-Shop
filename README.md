- **Клонирование репозитория с GitHub**
- **Запуск докер контейнера**
  <br/> `cd /docker`
  <br/> `docker-compose up`
- **Запуск проекта**
  <br/> `./gradlew bootRun --args='--spring.profiles.active=development'`
